# Platformer Game

A 2D platformer game project featuring a stick man character.

## Features

- Stick man player character
- Platform physics and collision detection
- Smooth movement mechanics

## Getting Started

### Prerequisites

- Node.js
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the Game

```bash
npm start
```
